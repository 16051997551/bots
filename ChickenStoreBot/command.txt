Разворачиваем на своём сайте виртуальное окружение virtualenv для Python:
1) Проверьте, какая версия интерпретатора установлена на вашем сервере:
python3 --version
2) Вы можете использовать последние версии виртуального окружения здесь: https://pypi.org/project/virtualenv/
3) Перейти в нужный каталог (вместо u и user укажите первую букву логина и сам логин):
cd /home/u/user/директория_сайта/public_html
4) Скачать и распаковать virtualenv:
wget https://pypi.python.org/packages/d4/0c/9840c08189e030873387a73b90ada981885010dd9aea134d6de30cd24cb8/virtualenv-15.1.0.tar.gz
tar xvfz virtualenv-15.1.0.tar.gz
5) Установить virtualenv:
python3 virtualenv-15.1.0/virtualenv.py myenv
6) В папке сайта /home/u/user/директория_сайта/public_html/ создайте файл .htaccess со следующим содержанием:
Options +ExecCGI
AddHandler cgi-script .py
7) Для активации локального виртуального окружения используется source: 
source /home/u/user/директория_сайта/public_html/myenv/bin/activate
8) Далее Вы можете приступить к установке библиотек с помощью команды pip

python -m pip install --upgrade pip
pip install flask
pip install viberbot
